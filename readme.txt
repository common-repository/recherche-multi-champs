Créer vos propres champs pour vos produits/articles/pages et proposer une recherche basée sur ces champs à vos visiteurs.

== Description ==

Donnez la possibilité aux visiteurs de votre site internet et vos clients de retrouver facilement vos produits, vos pages et vos articles.<br><br>
Pour cela, créer vos propres champs, associer les valeurs que vous souhaitez à ces champs et c’est parti! Grâce au formulaire de recherche que vous pouvez intégrer où vous le souhaitez via des shortcodes ou widgets, vos champs personnalisés sont la voie la plus rapide pour accéder à votre contenu.<br><br>
– Nombre de champs illimités,<br>
– Nombre de valeurs associées illimitées,<br>
– Champs de type textuel ou numérique,<br>
– Intégration facile du formulaire de recherche via Shortcode ou Widget,<br>
– Affichage des champs personnalisés directement dans les articles, pages et produits * via Shortcode,<br>
– Paramètres de personnalisation (Tailles, couleurs des formulaires de recherches), réglages divers (Possibilité de pas prendre en compte les pages, les articles, les produits * ou les champs vides…)<br><br>

== Installation ==

Pour installer le plugin: <br><br>
- Ouvrir le menu Extensions > Ajouter présent dans l'administration de votre site web.
- Cliquer sur "Parcourir", sélectionner le fichier du plugin avec l'extension ".zip" puis cliquer sur "Installer"
- Activer le plugin, disponible dans le menu "Extensions"
